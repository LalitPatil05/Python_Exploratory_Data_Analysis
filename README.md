# Exploratory Data Analysis 

# 🎯 📊 Diwali Sales Data Analysis Project

### 📝 Project Overview

**This project focuses on analyzing Diwali sales data using Python to uncover meaningful business insights. The objective was to understand customer purchasing behavior, identify high-performing customer segments, and determine which product categories generate the most sales.**

*Through exploratory data analysis (EDA), this project transforms raw sales data into actionable insights that can help businesses improve marketing strategies and increase revenue.*

---

### 🚀 Objectives

- Perform data cleaning and preprocessing

- Analyze customer demographics

- Identify high-performing states

- Understand purchasing patterns by age, gender, and profession

- Determine top-selling product categories

- Generate actionable business insights

---

### 🛠️ Tools & Technologies Used

- Python

- Pandas – Data manipulation & cleaning

- NumPy – Numerical operations

- Matplotlib – Data visualization

- Seaborn – Advanced visualizations

- Jupyter Notebook – Project development environment

---

### 📂 Dataset Description

- The dataset contains:

- Customer demographic details (Gender, Age Group, Marital Status, State)

- Profession/Occupation

- Product Category

- Purchase Amount

- Number of Orders

---

## Steps Performed

**1️⃣ Data Cleaning**

- Removed null values

- Dropped unnecessary columns

- Corrected data types

- Handled missing values

**2️⃣ Exploratory Data Analysis (EDA)**

- Gender-wise analysis

- Age group segmentation

- Marital status analysis

- State-wise order distribution

- Profession-wise sales analysis

- Product category analysis

**3️⃣ Data Visualization**

- Bar charts

- Count plots

- Aggregated sales comparisons

- Top 10 states visualization


## 📈 Key Insights

Married women contribute significantly to total purchases.
The age group 26–35 years shows the highest purchasing activity.
Customers from Uttar Pradesh, Maharashtra, and Karnataka generate the most orders.
Professionals working in IT, Aviation, and Healthcare sectors have strong purchasing behavior.

## Most popular product categories:
```
🛍Food
👗 Clothing
📱 Electronics
```


## ✅ Final Conclusion

***Married women aged 26–35 years, residing in Uttar Pradesh, Maharashtra, and Karnataka, and working in the IT, Aviation, and Healthcare sectors are more likely to purchase products, particularly from the Food, Clothing, and Electronics categories.***

## 💼 Business Recommendations

- Focus marketing campaigns on high-performing states.
- Target advertisements towards married women aged 26–35.
- Promote Food, Clothing, and Electronics categories during festive seasons.
- Create personalized offers for IT, Aviation, and Healthcare professionals.
